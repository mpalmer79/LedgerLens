# LedgerLens icon inventory

All icons are from [lucide-react](https://lucide.dev/) at stroke-width 1.5–2, sizes 14–28px. Two bespoke SVGs (`logomark.svg`, `ledger-motif.svg`) live in this folder.

## Currently used

- `Wallet` — accounts / chart of accounts
- `Receipt` — individual transaction
- `TrendingUp` — confidence-up indicator
- `BarChart3` — eval dashboard / metrics
- `CheckCircle2` — auto-categorized status
- `AlertTriangle` — review queue / low confidence
- `HelpCircle` — UNCATEGORIZABLE
- `Search` — transaction search
- `Filter` — table filters
- `Download` — export
- `Calendar` — date range
- `DollarSign` — amount field
- `ChevronRight` — disclosure / nav
- `Copy` — clipboard
- `ExternalLink` — outbound link
- `Loader2` — loading spinner (`animate-spin`)
- `RefreshCw` — refresh action

## Reserved (planned, not yet in product)

- `ShieldCheck` — calibration validated
- `Fingerprint` — audit trail
