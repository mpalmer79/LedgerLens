# LedgerLens — Backend

FastAPI service implementing the LedgerLens ingest, categorization, routing, and corrections pipeline. Stack: Python 3.12, FastAPI, SQLAlchemy + Alembic, Postgres with pgvector, Anthropic SDK.

See the [repo root README](../README.md) for setup instructions and [docs/ARCHITECTURE.md](../docs/ARCHITECTURE.md) for design rationale.
